## gnas.dev

Personal tech ecosystem by a single developer based in Ho Chi Minh City, Vietnam. Self-hosted applications running on a K3s Kubernetes cluster with GitOps, Authentik SSO, and full observability. Built with Go, Kotlin Multiplatform, Angular, React, and modern cloud-native infrastructure. Everything from code to production, by one person.
